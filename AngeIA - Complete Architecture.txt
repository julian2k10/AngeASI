## Debiasing Strategies

Confidence-Weighted Routing (Language Router)

Each expert declares confidence (0-1) independent of training history
Only high-confidence experts get routed to
Prevents routing inertia and training-order bias


Task-Aware Shared Layer Routing (Shared Experts)

Routes based on linguistic phenomena analysis, not input distribution
Family info has weak 0.1x influence (hints but doesn't dominate)
Prevents shared layer from being captured by first-trained language


Stratified Load Balancing (Critical!)

Balance expert load within each language, not globally
Prevents high-resource languages from forcing imbalance
Ensures low-resource experts aren't overloaded


Specialization Monitoring (Early Detection)

Track how each expert routes across languages
Detect routing bias before it damages performance
Metrics: expert specialization, max trend, dominance

## Token Partitioning Within Language Expert

```python
# Language expert for, say, English
english_expert = {
    'shared_components': SharedFFN(hidden_dim),
    'sub_experts': [
        SubExpert(25k tokens: 0-25k),        # Expert 1
        SubExpert(25k tokens: 25k-50k),      # Expert 2
        SubExpert(25k tokens: 50k-75k),      # Expert 3
        # ... up to 8 experts depending on resource level
    ],
    'consolidation_layer': SharedFFN(hidden_dim)  # Merges sub-expert outputs
}
```

**Flow within a language expert:**

```python
# Input: hidden states from Family Expert
x = family_expert_output  # [batch, seq_len, hidden_dim]

# Shared processing
x_shared = english_expert['shared_components'](x)

# Identify which tokens belong to which sub-expert
token_ids = input_tokens  # [batch, seq_len]
token_partition = token_ids // 25000  # Which partition: 0, 1, 2, ..., 7

# Route tokens to corresponding sub-experts
output = torch.zeros_like(x_shared)
for partition_idx in range(num_partitions):
    mask = (token_partition == partition_idx)
    if mask.sum() > 0:  # Only activate if tokens exist
        x_partition = x_shared[mask]
        output[mask] = english_expert['sub_experts'][partition_idx](x_partition)

# Consolidate
output = english_expert['consolidation_layer'](output)
```

## Why This Works Beautifully

**1. Perfect load balancing across token space**
- Tokens 0-25k always route to Expert 1 (deterministic)
- No load imbalance within a language
- Each sub-expert sees consistent token distribution

**2. Dynamic computation**
- English document with mostly tokens 0-100k? Only Expert 1-4 activate
- Domain-specific text using rare tokens? Only relevant experts activate
- No wasted computation on unused token ranges

**3. Resource-aware allocation**
- High-resource languages (English, Mandarin): 8 sub-experts (200k tokens)
- Medium-resource (Spanish, French): 4 sub-experts (100k tokens)
- Low-resource (Icelandic, Amharic): 2 sub-experts (50k tokens)

This is far more efficient than giving all languages the same parameter budget.

**4. Per-family consolidation**
- All active language experts in Indo-European consolidate through shared layer
- This maintains coherence across English + Spanish mixing
- Prevents independent language experts from diverging

## Load Balancing & Consolidation

**Per-language consolidation:**
```python
def apply_language_expert(x, lang_expert, token_ids):
    x_shared = lang_expert['shared'](x)
    
    outputs = []
    for partition_idx in range(num_partitions):
        mask = (token_ids // token_slice == partition_idx)
        if mask.sum() > 0:
            x_part = x_shared[mask]
            out_part = lang_expert['sub_experts'][partition_idx](x_part)
            outputs.append((mask, out_part))
    
    # Merge all sub-expert outputs
    merged = torch.zeros_like(x_shared)
    for mask, out in outputs:
        merged[mask] = out
    
    # Consolidation ensures coherence
    final = lang_expert['consolidation_layer'](merged)
    return final
```

## Implementation Pattern

```python
class DynamicVocabOutputLayer(nn.Module):
    def __init__(self, hidden_dim, full_vocab_size):
        super().__init__()
        self.output_layer = nn.Linear(hidden_dim, full_vocab_size)
    
    def forward(self, hidden_states, active_vocab_ids):
        """
        hidden_states: [batch, seq_len, hidden_dim]
        active_vocab_ids: list of token IDs to include in softmax
        """
        # Compute logits for full vocab
        full_logits = self.output_layer(hidden_states)  # [batch, seq_len, full_vocab]
        
        # Slice to active vocabulary
        active_logits = full_logits[..., active_vocab_ids]  # [batch, seq_len, active_size]
        
        # Softmax only over active
        log_probs = torch.log_softmax(active_logits, dim=-1)  # [batch, seq_len, active_size]
        
        return log_probs, active_vocab_ids
    
    def get_active_vocab(self, active_languages, active_sub_experts, 
                        lang_token_ranges, tokens_per_expert=25000):
        """
        Build active vocab list based on which languages/sub-experts are active
        """
        active_vocab_ids = []
        
        for lang in active_languages:
            lang_start = lang_token_ranges[lang]['start']
            
            for sub_idx in active_sub_experts[lang]:
                token_start = lang_start + (sub_idx * tokens_per_expert)
                token_end = token_start + tokens_per_expert
                active_vocab_ids.extend(range(token_start, token_end))
        
        return sorted(active_vocab_ids)
```

## Loss Computation

```python
def compute_loss(log_probs, target_token_ids, active_vocab_ids, active_vocab_mapping):
    """
    log_probs: [batch, seq_len, active_vocab_size]
    target_token_ids: [batch, seq_len] (global token IDs)
    active_vocab_mapping: dict mapping global_id → active_idx
    """
    batch_size, seq_len = target_token_ids.shape
    
    # Map target tokens to active vocab indices
    active_target_indices = torch.zeros_like(target_token_ids)
    
    for b in range(batch_size):
        for t in range(seq_len):
            global_id = target_token_ids[b, t].item()
            if global_id in active_vocab_mapping:
                active_target_indices[b, t] = active_vocab_mapping[global_id]
            else:
                # Target token not in active vocab—this shouldn't happen in well-designed system
                # But if it does, skip or use OOV token
                active_target_indices[b, t] = 0  # or handle differently
    
    # Standard cross-entropy over active vocab
    loss = torch.nn.functional.nll_loss(
        log_probs.reshape(-1, log_probs.size(-1)),
        active_target_indices.reshape(-1),
        reduction='mean'
    )
    
    return loss
```

```python
def forward_with_dynamic_vocab(model, input_ids):

    families = detect_families(input_ids)

    x = model.family_experts[families[0]](x)

    outputs = []
    active_sub_experts = {}
    
    for lang in active_languages:
        lang_output, active_subs = model.language_experts[lang](x, input_ids)
        outputs.append(lang_output)
        active_sub_experts[lang] = active_subs
    
    # 5. Consolidate
    x = consolidate_language_outputs(outputs)
    
    # 6. **Build active vocabulary**
    active_vocab_ids = build_active_vocab(
        active_languages, 
        active_sub_experts,
        model.lang_token_ranges
    )
    
    # 7. **Dynamic vocab softmax**
    log_probs, active_vocab = model.output_layer(x, active_vocab_ids)
    
    # 8. Sampling/greedy
    predictions = torch.argmax(log_probs, dim=-1)
    
    # 9. **Map back to global token IDs**
    global_predictions = torch.zeros_like(predictions)
    for idx, active_id in enumerate(active_vocab_ids):
        mask = (predictions == idx)
        global_predictions[mask] = active_id
    
    return global_predictions
```

## Critical Detail: OOV Handling

```python
class SafeDynamicVocabOutputLayer(nn.Module):
    def __init__(self, hidden_dim, full_vocab_size, unk_token_id=0):
        super().__init__()
        self.output_layer = nn.Linear(hidden_dim, full_vocab_size)
        self.unk_token_id = unk_token_id
    
    def forward(self, hidden_states, active_vocab_ids, target_ids=None):
        full_logits = self.output_layer(hidden_states)
        active_logits = full_logits[..., active_vocab_ids]
        log_probs = torch.log_softmax(active_logits, dim=-1)
        
        if target_ids is not None and self.training:
            # Check for OOV tokens
            valid_mask = torch.isin(target_ids, torch.tensor(active_vocab_ids))
            if not valid_mask.all():
                # Replace OOV targets with UNK
                target_ids = torch.where(valid_mask, target_ids, self.unk_token_id)
        
        return log_probs
```

```python
class DynamicVocabOutputLayer(nn.Module):

        self.output_layer = nn.Linear(hidden_dim, full_vocab_size)
    
    def forward(self, hidden_states, active_vocab_ids):
        """
```
